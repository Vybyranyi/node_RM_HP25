# Node RM HP25

A professional Node.js backend application designed for resource management and property automation. This system provides a structured API to handle bookings, inventory, and user permissions efficiently.

## 📂 Project Structure

The project follows a modular architecture for better scalability:

* **src/config**: Configuration files (Database, Environment).
* **src/controllers**: Logic for handling requests and returning responses.
* **src/middleware**: Security, authentication, and validation layers.
* **src/models**: Data schemas and database interactions.
* **src/routes**: API endpoint definitions.
* **src/services**: Core business logic.
* **src/utils**: Common utility functions and constants.

## 🚀 Features

* **RESTful Architecture**: Clean and predictable API endpoints.
* **Secure Auth**: Implementation of JWT for protected routes.
* **Modular Design**: Easy to maintain and extend.
* **Validation**: Strict input validation for data integrity.

## 🛠 Tech Stack

* **Runtime**: Node.js
* **Framework**: Express.js
* **Database**: MySQL
